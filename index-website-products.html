Cài đặt Supabase

Không dán service role key. Chỉ dùng anon key. Nếu chưa kết nối Supabase, app lưu tạm localStorage để test.

SQL tạo bảng Supabase

Hướng dẫn

  1. Chạy SQL tạo bảng trong Supabase.
  2. Tạo bucket Storage public tên product-images.
  3. Vào Cài đặt Supabase, nhập URL + anon key.
  4. Nhập sản phẩm, upload ảnh đại diện, upload gallery.
  5. Bấm lưu, sau đó xuất CSV WooCommerce để import vào WordPress.
